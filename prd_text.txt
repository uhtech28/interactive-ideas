IMPLEMENTATION NOTE

At Stage 1 start: brightness = 0% + 0% = 0% (darkest state).
After completing Stage 1 fully and entering Stage 2: brightness = 8.57% + 0% = 8.57%.
Mid-Stage 5 with 50% of tasks done: brightness = (4 x 8.57%) + (50% x 40%) = 34.28% + 20% = 54.28%.
Final stage fully complete: brightness = (7 x 8.57%) + 40% = 60% + 40% = 100%.
Brightness drives a single Phaser post-processing filter (brightness/contrast). No per-biome custom logic
required.

•

Displayed in HUD as a running project metric throughout the project lifespan.
Always increases — quality determines magnitude of increase per stage advance, never direction.
